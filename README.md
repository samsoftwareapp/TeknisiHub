# TeknisiHub

TeknisiHub adalah software yang dikembangkan oleh **samsoftwareapp** untuk mendukung kebutuhan akses layanan, tools, dan workflow teknisi.

Live page:
`https://samsoftwareapp.github.io/TeknisiHub/`

## Tentang Project

TeknisiHub merupakan web interface untuk membantu teknisi mengakses layanan dan tools secara lebih terpusat. Project ini menggabungkan antarmuka web dengan local service agar proses akses tool, login, dan workflow teknis bisa berjalan lebih praktis.

## Fitur Utama

- Login akses menggunakan nomor Telegram dan verifikasi kode.
- Integrasi dengan local service untuk membaca status koneksi dan versi service.
- Dashboard akses member untuk kebutuhan BIOS, Boardview, dan tools teknisi.
- SPI Flash workbench berbasis Web UI untuk proses read, write, erase, detect, dan verify.
- Katalog file dan tools lokal untuk kebutuhan teknisi.

## Struktur Project

- `index.html` untuk tampilan utama aplikasi.
- `styles.css` untuk styling antarmuka.
- `app.js` untuk logika utama aplikasi.
- `pages/spi-flash/page.js` untuk halaman dan logic SPI Flash.
- `assets/` untuk kebutuhan aset visual.

## Penggunaan

1. Buka halaman live di `https://samsoftwareapp.github.io/TeknisiHub/`
2. Pastikan local service TeknisiHub aktif bila fitur backend lokal ingin digunakan.
3. Login menggunakan nomor Telegram.
4. Setelah akses aktif, gunakan menu layanan atau tools sesuai izin akses yang tersedia.
